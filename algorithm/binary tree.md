# 概念

二叉树（Binary Tree），是指有两个分支的树结构，二叉树是有n（n>=0）个有限元素的集合，若n=0则该二叉树称为空二叉树，若n>0时，根节点和两个互不相交的节点被称为左子树和右子树，这两个数同时也时一个二叉树。

五种基本形态

1. 空二叉树
2. 只有一个根节点
3. 只有左子树
4. 只有右子树
5. 完全二叉树

## 满二叉树

如果一个二叉树的第 $i$ 层有两个 $2^{i-1}$ 个节点的二叉树成为 满二叉树（根节点所在深度 $k_0=1$ ）

性质：有 $2^k-1$ 个节点，节点个数一定为技术，有 $2^{k-1}个叶子$

![img](https://blog.icecliffs.cn/files/img/2020/12/7a20e22e71db890c548741f6466b408b.jpg)

总节点（k）：$k=2^h-1$

树高（h）：$h=log_2(k+1)$

## 完全二叉树

设深度为 $h$ 有 $i$ 个结点的二叉树，且当其中的每一节点都可以和相同深度的满二叉树中编号为1~i的节点一一对应时，称为**完全二叉树**

![img](https://blog.icecliffs.cn/files/img/2020/12/43075b4863bc00e53a38ae81762d2a26.jpg)

完全二叉树

总节点（k）：$2^{h-1}<=k<=2^h-1$

树高（h）：$h=log_2k+1$
